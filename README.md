# 网络计算器 v1.0

## 简介

网络计算器是一款基于 PyQt5 开发的图形化网络工具，支持 IPv4 地址与子网掩码的多种格式输入，能够快速计算网络地址、广播地址、可用主机范围以及子网划分（包括 VLSM 变长子网掩码）。该工具还内置了版本更新检测功能。

## 功能特点

- **多格式输入**：支持二进制、十进制、十六进制、点分十进制及简写（如 `/24`）格式的 IP 地址和子网掩码输入。
- **网络信息计算**：显示 IP 地址和子网掩码的多种进制表示，输出网络地址、广播地址、可用主机范围及主机数量。
- **子网划分**：
  - 根据子网数量划分子网
  - 根据主机数量划分子网（VLSM）
- **输出美化**：支持 HTML 格式输出，便于阅读和复制。
- **版本更新**：自动检测 GitHub 仓库中的最新版本，并显示更新日志。
- **帮助与关于**：提供版本信息、作者联系方式及官网链接。

## 运行环境

- Python 3.6 或更高版本
- PyQt5
- requests

## 使用说明
输入格式
IP 地址：支持以下格式：
点分十进制：192.168.1.1
十进制整数：3232235777
十六进制（8 位）：C0A80101
二进制（32 位）：11000000101010000000000100000001

子网掩码：支持以下格式：
点分十进制：255.255.255.0
十进制整数：4294967040
十六进制（8 位）：FFFFFF00
二进制（32 位）：11111111111111111111111100000000
简写：24（表示 /24）

## 功能操作
计算：根据输入的 IP 地址和子网掩码，计算网络信息。
VLSM：根据子网数量或主机数量，划分子网。
清除：清空所有输入和输出内容。

## 更新功能
程序启动后可通过“帮助”菜单中的“更新”选项检查新版本。版本信息与更新日志从 GitHub 仓库获取，支持自动提示和官网下载跳转。
