# 子网掩码计算器 v1.1

一款功能强大的网络管理工具，集成了子网掩码计算、VLSM可变长子网划分以及实用的网络工具集（Ping、端口扫描、DNS查询）。

## 📋 功能特性

### 1. 子网掩码计算
- ✨ **IP地址格式转换** - 支持二进制、十进制、十六进制、点分十进制之间的相互转换
- 🔧 **子网信息计算** - 计算网络地址、广播地址、可用IP范围、可用IP数量
- 📝 **支持多种输入格式** - 点分十进制、CIDR简写(/24)、二进制(32位)、十六进制、十进制整数
- 🎨 **友好的可视化界面** - 彩色分区显示，结果清晰易读

### 2. VLSM可变长子网划分
- 📊 **按子网数量划分** - 根据指定的子网数量自动计算新掩码
- 💻 **按主机数量划分** - 根据每个子网所需主机数量自动划分
- 📋 **表格化展示** - 以表格形式清晰展示所有子网信息

### 3. 网络工具集

#### 🏓 Ping工具
- 支持单IP、域名、IP范围（如：192.168.1.1-192.168.1.10）
- 支持CIDR格式（如：192.168.1.0/28）
- 支持批量Ping（每行一个目标）
- 可设置Ping次数和超时时间
- 显示平均延迟时间

#### 🔍 端口扫描
- TCP/UDP端口扫描
- 自定义端口范围
- 常用端口快捷按钮（Web、SSH、FTP、MySQL等）
- 并发扫描提高速度
- 显示端口开放/关闭状态

#### 📡 DNS查询
- 支持A、AAAA、MX、NS、TXT、CNAME记录
- 域名解析功能
