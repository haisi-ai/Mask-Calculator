# 子网掩码计算器

## 简介

子网掩码计算器是一款基于 PyQt5 开发的图形化网络工具，专门用于 IPv4 地址和子网掩码的计算与子网划分。该工具支持多种输入格式，能够快速计算网络地址、广播地址、可用主机范围，并支持基于子网数量或主机数量的 VLSM（变长子网掩码）划分。

## 主要特性

### 多格式输入支持
- **IP 地址**：支持点分十进制、十进制整数、十六进制（8位）、二进制（32位）格式
- **子网掩码**：支持点分十进制、十进制整数、十六进制（8位）、二进制（32位）、简写格式（如 /24）

### 核心功能
- **网络信息计算**：
  - IP 地址和子网掩码的多种进制表示（二进制、十六进制、十进制）
  - 网络地址和广播地址
  - 可用主机范围及主机数量
  - 子网掩码简写形式

- **VLSM 子网划分**：
  - 根据指定的子网数量自动划分子网
  - 根据每个子网需要的主机数量自动划分子网
  - 输出包含网络地址、子网掩码、主机范围、广播地址的表格形式

### 用户体验
- **友好的界面**：清晰的布局，彩色输出，易于阅读
- **预设选项**：常用 IP 地址和子网掩码快速选择
- **一键清除**：快速清空所有输入和输出
- **HTML 格式输出**：支持彩色显示，便于复制和分享

### 更新功能
- **自动检查更新**：从 GitHub 仓库获取最新版本信息
- **更新日志展示**：显示版本更新内容
- **网络状态检测**：检查网络连接状态

## 系统要求

- **操作系统**：Windows / Linux / macOS
- **Python 版本**：Python 3.6 或更高版本
- **依赖库**：
  - PyQt5

## 使用说明
基本计算流程
输入 IP 地址：支持多种格式，可直接输入或从下拉列表选择
输入子网掩码：支持多种格式，可直接输入或从下拉列表选择
点击"计算"按钮：显示网络信息计算结果
VLSM 子网划分流程

方式一：按子网数量划分
输入 IP 地址和子网掩码
在"子网数量"输入框中输入需要划分的子网数量（如 4、8、16 等）
点击"VLSM"按钮
查看划分后的子网列表

方式二：按主机数量划分
输入 IP 地址和子网掩码
在"主机数量"输入框中输入每个子网需要的主机数量（可用 IP 数量）
点击"VLSM"按钮
查看划分后的子网列表
注意：当同时填写子网数量和主机数量时，程序优先按子网数量进行划分。

输入格式示例
类型	格式示例	说明
点分十进制 IP	192.168.1.1	标准 IPv4 地址格式
十进制整数 IP	3232235777	IP 地址的整数表示
十六进制 IP	C0A80101	8位十六进制数
二进制 IP	11000000101010000000000100000001	32位二进制数
点分十进制掩码	255.255.255.0	标准子网掩码格式
简写掩码	24	前缀长度表示法
十进制整数掩码	4294967040	子网掩码的整数表示
十六进制掩码	FFFFFF00	8位十六进制数
二进制掩码	11111111111111111111111100000000	32位二进制数

## 常见问题
Q: 为什么点击"计算"后没有反应？
A: 请检查 IP 地址和子网掩码输入是否正确，程序会在输出框中显示具体的错误信息。

Q: VLSM 划分时提示"子网数量无效"？
A: 请确保输入的子网数量为正整数，且不超过原始网络的子网划分上限。

Q: 更新功能提示"无法连接到网络"？
A: 请检查您的网络连接，确保能够访问 GitHub。如果网络正常，可能是防火墙或代理设置问题。

Q: 程序图标无法显示？
A: 请确保 logo.ico 文件与程序在同一目录下。
