# 子网掩码计算器 v1.1

一款功能强大的网络管理工具，集成了子网掩码计算、VLSM可变长子网划分以及实用的网络工具集（Ping、端口扫描、DNS查询）。

## 📋 功能特性

### 1. 子网掩码计算
- ✨ **IP地址格式转换** - 支持二进制、十进制、十六进制、点分十进制之间的相互转换
- 🔧 **子网信息计算** - 计算网络地址、广播地址、可用IP范围、可用IP数量
- 📝 **支持多种输入格式** - 点分十进制、CIDR简写(/24)、二进制(32位)、十六进制、十进制整数
- 🎨 **友好的可视化界面** - 彩色分区显示，结果清晰易读

### 2. VLSM可变长子网划分
- 📊 **按子网数量划分** - 根据指定的子网数量自动计算新掩码
- 💻 **按主机数量划分** - 根据每个子网所需主机数量自动划分
- 📋 **表格化展示** - 以表格形式清晰展示所有子网信息

### 3. 网络工具集

#### 🏓 Ping工具
- 支持单IP、域名、IP范围（如：192.168.1.1-192.168.1.10）
- 支持CIDR格式（如：192.168.1.0/28）
- 支持批量Ping（每行一个目标）
- 可设置Ping次数和超时时间
- 显示平均延迟时间

#### 🔍 端口扫描
- TCP/UDP端口扫描
- 自定义端口范围
- 常用端口快捷按钮（Web、SSH、FTP、MySQL等）
- 并发扫描提高速度
- 显示端口开放/关闭状态

#### 📡 DNS查询
- 支持A、AAAA、MX、NS、TXT、CNAME记录
- 域名解析功能

## 🚀 安装与运行

### 环境要求
- Python 3.6+
- Windows / Linux / macOS

### 安装依赖

```bash
pip install PyQt5 requests ipaddress
可选依赖（用于完整DNS功能）
bash
pip install dnspython
运行程序
bash
python 子网掩码计算器v1.1.py
📖 使用指南
子网掩码计算
在"IP地址配置"区域输入IP地址和子网掩码

支持多种输入格式：

点分十进制：192.168.1.1

CIDR简写：/24

二进制：11000000101010000000000100000000

十六进制：C0A80100

十进制整数：3232235776

点击"计算子网信息"按钮

查看计算结果

VLSM子网划分
输入基础网络地址和子网掩码

选择划分方式：

输入子网数量

或输入每个子网所需主机数量

点击"VLSM子网划分"按钮

查看划分结果表格

Ping工具
点击菜单栏：工具 → Ping工具

输入目标地址（支持多种格式）

设置Ping次数和超时时间

点击"开始Ping"

查看实时结果

端口扫描
点击菜单栏：工具 → 端口扫描

输入目标IP地址或域名

设置端口范围或使用常用端口快捷按钮

选择扫描类型（TCP/UDP）

点击"开始扫描"

DNS查询
点击菜单栏：工具 → DNS查询

输入域名

选择记录类型

点击"查询"

⚠️ 注意事项
端口扫描：大规模端口扫描可能耗时较长，建议使用常用端口快捷按钮

UDP扫描：UDP扫描结果可能不如TCP扫描准确

MX/NS记录：需要安装dnspython库才能完整支持

网络连接：更新检查和某些DNS查询需要互联网连接

🔧 常见问题
Q: 程序无法启动？
A: 请确保已安装所有依赖：pip install PyQt5 requests ipaddress

Q: Ping工具显示"超时"？
A: 请检查目标主机是否在线，或适当增加超时时间

Q: DNS查询MX记录失败？
A: 需要安装dnspython库：pip install dnspython

Q: 端口扫描结果不准确？
A: 某些端口可能有防火墙保护，UDP扫描结果仅供参考
